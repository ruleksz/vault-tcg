# TCG Price Tracker

A modern, real-time Trading Card Game price tracking website built with React, TypeScript, and Tailwind CSS.

## Features

- 🎴 **Real-time Price Updates**: Live price updates every 3 seconds with animated price changes
- 🔍 **Multi-TCG Support**: Track prices for Pokémon, Yu-Gi-Oh!, Magic: The Gathering, and One Piece cards
- 💎 **Glassmorphism Design**: Modern, premium UI with neon glow effects
- 📊 **Price Charts**: Interactive price history charts for each card
- 🔎 **Search & Filter**: Find cards by name, rarity, or price range
- 📱 **Responsive Design**: Fully responsive mobile-first design
- ✨ **Smooth Animations**: Framer Motion animations throughout the app

## Tech Stack

- **React 18** with TypeScript
- **Vite** for fast builds
- **Tailwind CSS** for styling
- **Framer Motion** for animations
- **Recharts** for price charts
- **TanStack Query** for data fetching
- **Axios** for API calls

## Getting Started

### Prerequisites

- Node.js 16+ and npm/yarn/pnpm

### Installation

1. Clone the repository:

```bash
git clone https://github.com/ruleksz/vault-tcg.git
cd vault-tcg
npm install && npm run dev
```
